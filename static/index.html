🎭 MyDramaList Scraper API

✅ API Status: Online
A serverless API for scraping MyDramaList.com data

📚 Available Endpoints

GET
/api/search/q/{query}
Search for dramas by query
Example: /api/search/q/my%20demon
GET
/api/id/{slug}
Get drama details by slug
Example: /api/id/746993-my-demon
GET
/api/id/{slug}/cast
Get cast information for a drama
Example: /api/id/746993-my-demon/cast
GET
/api/id/{slug}/episodes
Get episode details for a drama
Example: /api/id/746993-my-demon/episodes
GET
/api/id/{slug}/reviews
Get reviews for a drama
Example: /api/id/746993-my-demon/reviews
GET
/api/people/{people_id}
Get person details by ID
Example: /api/people/14472-song-kang
GET
/api/seasonal/{year}/{quarter}
Get seasonal dramas (quarter: 1-4)
Example: /api/seasonal/2023/4
GET
/api/list/{id}
Get a specific drama list by ID
Example: /api/list/p1R8nrJ3
GET
/api/dramalist/{user_id}
Get a user's drama list by user ID
Example: /api/dramalist/CHOCOLATE
GET
/api/health
Health check endpoint

📝 Response Format

All endpoints return JSON responses. Error responses follow this format:

{"code": 404, "error": true, "description": "404 Not Found"}

⚠️ Rate Limiting

This API includes built-in rate limiting (1 second delay between requests) to respect MyDramaList.com's servers. Please use responsibly.
